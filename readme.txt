Game Sales Analysis
===================

This program allows you to filter and analyze video game sales data.

### Requirements
- Python 3.7 or higher
- `pandas` library (run `pip install pandas` if not installed)

### Instructions
1. Unzip the folder.
2. Make sure `game_sales_analysis.py` and `vgsales.csv` are in the same folder.
3. Open a terminal or command prompt.
4. Run the script:
   python game_sales_analysis.py
